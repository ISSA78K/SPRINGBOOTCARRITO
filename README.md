# Carrito de Compras Spring Boot

Proyecto monolítico desarrollado con Spring Boot y JPA.

## Tecnologías utilizadas
- Java
- Spring Boot
- Spring Data JPA
- H2 Database
- Postman

## Funcionalidades
- Crear productos
- Crear carrito
- Agregar productos al carrito
- Ver productos
- Ver items del carrito
- Calcular total del carrito

## Colección Postman
La colección se encuentra en:
